# Python Calculator
A simple Python-based calculator that performs basic arithmetic operations using user input from the terminal.
This project is designed as a beginner-friendly Python task and demonstrates basic concepts such as input handling, conditional statements, and arithmetic operations.
# Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Handles division by zero safely
- Clear and readable output
# Technologies Used
- Python 3
(No external libraries required)
